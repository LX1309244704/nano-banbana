# Nano-Banana AI 图像创作工作室

一款基于 React + TypeScript 的现代化 AI 图像创作应用，支持图像生成、编辑、背景移除和视频生成功能。

## 🎨 核心功能

### 🤖 AI 图像生成
- **智能生成**：使用 Nano-Banana 优化模型，支持高分辨率图像生成
- **多比例支持**：1:1、16:9、9:16、4:3、3:4 等多种比例
- **高质量输出**：基于 Gemini 2.5 Flash Image Preview 模型优化

### 🎭 图层编辑
- **多层管理**：背景层、叠加层、视频层等完整图层系统
- **实时编辑**：拖拽、缩放、旋转、透明度调整
- **混合模式**：正常、正片叠底、滤色、叠加等专业混合模式

### 🖌️ 智能编辑
- **遮罩重绘**：精确绘制遮罩区域进行局部重绘
- **背景移除**：一键智能移除图像背景
- **图像合成**：多图层智能合成，保持光影一致性

### 🎬 AI 视频生成
- **文本转视频**：基于 Sora-2 模型生成高质量视频
- **多时长选择**：支持 10秒、15秒 视频生成
- **图片引导**：可上传参考图像生成相关视频内容

## 🚀 技术特性

### 前端技术栈
- **React 18** + **TypeScript** - 现代化前端框架
- **Vite** - 快速的构建工具
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Lucide React** - 精美的图标库

### AI 集成
- **Nano-Banana API** - 专为图像生成优化的模型
- **Gemini 2.5 Flash** - Google 最新视觉模型
- **Sora-2** - OpenAI 视频生成技术

### 数据管理
- **IndexedDB** - 本地高分辨率图像缓存
- **LocalStorage** - 用户配置和历史记录持久化

## 🛠️ 快速开始

### 环境要求
- Node.js 16+ 
- npm 或 yarn

### 安装运行
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

应用将在 `http://localhost:3000` 启动

### API 配置
1. 点击右上角设置按钮
2. 输入您的 API Key
3. 支持 Gemini 和 OpenAI 格式的 API Key

## 📱 使用指南

### 图像生成
1. 选择"生成"模式
2. 输入描述性提示词
3. 选择合适比例
4. 点击生成按钮

### 图层编辑
1. 上传或生成背景图像
2. 添加叠加图层
3. 调整位置、缩放、透明度
4. 使用混合模式优化效果

### 视频生成
1. 切换到"视频"模式
2. 输入视频描述
3. 选择时长和比例
4. 可上传参考图像
5. 点击生成按钮

### 历史管理
- **查看历史**：右侧面板显示所有生成记录
- **添加图层**：点击历史作品可快速添加到画布
- **删除作品**：悬停后点击删除按钮移除作品

## 🚀 部署指南

### Vercel 部署
```bash
npm install -g vercel
vercel --prod
```

### Netlify 部署
```bash
npm run build
# 将 dist 目录部署到 Netlify
```

### Docker 部署
```dockerfile
FROM node:16-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["npm", "run", "preview"]
```

## 💡 使用技巧

### 提示词优化
- 使用具体描述而非抽象概念
- 包含光影、风格、构图等信息

- 背景层应放置在最底层

### 视频生成
- 描述应包含动作和场景变化
- 参考图像可帮助模型理解风格
- 选择合适的时长和比例
